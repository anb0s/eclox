About

Eclox is a simple doxygen frontend plug-in for eclipse.

Eclox provides a slim and sleek integration of the code documentation process into eclipse. Features include graphical edition of doxyfile settings, integrated doxygen invokation and doxygen outputs logging.

Eclox is a free software distributed under the term of the GNU General Public Licence (GPL).

Download & Install

Requirements

Latest eclox releases needs JRE 1.4.2 (or above) and eclipse 3.1 (or above). Eclox has not been officaly tested on GNU class-path yet.

Doxygen must be installed on your system in order to generate documentation but is not required to simply edit doxyfiles.

Update site

The more convenient and the prefered way to install eclox is to use eclipse's update manager (learn more) and the update site at the address below.

http://download.gna.org/eclox/update

Binary Package

Note: since eclox 0.5.4 packages contain a feature.

Download eclox 0.5.4 binary package.

Uninstall or disable any existing eclox version (learn more) and leave all eclipse instances. Extract the package content in eclipse installation folder (or alternatively in your personnal plugin and feature repository). Finally (re-)start eclipse and you are done.

You can still get older eclox releases for previous eclipse versions.

Sources

Sources of the software can be retrieved from the CVS repository. See following command sample for details about how to connect to the repository.

cvs -d :pserver:anonymous@cvs.gna.org:/cvs/eclox co eclox
