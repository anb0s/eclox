About

Eclox is a simple doxygen frontend plug-in for eclipse. It aims to provide a slim and sleek integration of the code documentation process into eclipse.

Eclox is a free software distributed under the term of the GNU General Public Licence (GPL).

Features Overview

  • Graphical edition of doxyfile settings,
  • integrated doxygen invokation,
  • doxygen outputs logging and
  • packaged doxygen binaries (for windows and linux).

Download & Install

Latest eclox releases needs JRE 1.4.2 (or above) and eclipse 3.1 (or above). Eclox has not been officaly tested on GNU class-path yet, so feedback is welcome.

Important! Prior to eclox 0.6.0, doxygen must be installed on your system in order to generate documentation but is not required to simply edit doxyfiles.

Update site

The more convenient and the prefered way to install eclox is to use eclipse's update manager (learn more) and the update site at the address below.

http://download.gna.org/eclox/update

Binary Package

  1. Uninstall or disable any existing eclox version (learn more) and leave all eclipse instances.
  2. Download eclox 0.6.0 binary package (for all platformes) and optionnaly packaged doxygen 1.5.0 binaries (for windows and linux).
  3. Extract the package content in eclipse installation folder (or alternatively in your personnal plugin and feature repository).
  4. Finally (re-)start eclipse and you are done.

You can still get older eclox releases for previous eclipse versions.

Getting Involved

Bugs

Bugs in eclox should be reported in the dedicated online tool hosted by Gna!. Submiters should also supply a valid email address in order to receive updates.

Mailing lists

For discussion about eclox, subscribe to the user mailing list or browse the online archives. For announces about eclox, subscribe to the news mailing list.

Source Code

The project page holds informations about developments. Sources of the software can be retrieved from the CVS repository. See following command sample for details about how to connect to the repository.

cvs -d :pserver:anonymous@cvs.gna.org:/cvs/eclox co eclox
